Struct pikadick_system_info::CacheContext

source · 
pub struct CacheContext { /* private fields */ }
Expand description

A context for caching data related to information queries.

A system may occasionally give more data than requested when asked. Using this allows the use of that data without fetching it again. A new context should be created if new data is required, as it will happily reuse old data if it has it.
